# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-05

### Added
- Initial release
- Placeholder constants for UI components
- TypeScript type definitions
- `getPlaceholder()` utility function
- `getAllPlaceholders()` utility function
- Full TypeScript support with `PlaceholderKey` type

### Placeholders Included
- `SELECT_OPTION` - Select component placeholder
- `FILTER_OPERATOR` - Filter operator field
- `FILTER_MIN` - Minimum value input
- `FILTER_MAX` - Maximum value input
- `FILTER_SELECT_VALUE` - Single value selection
- `FILTER_SELECT_VALUES` - Multiple values selection
- `FILTER_SELECT_DATE` - Date picker input
- `FILTER_VALUE` - Generic value input
- `FILTER_SELECT_FIELD` - Field selection dropdown
- `SEARCH` - Search input placeholder
- `INPUT_EMPTY` - Empty placeholder
